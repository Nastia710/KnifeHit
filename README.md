# KnifeHit

# Ціль
Є 2 види цілі: для звичайних рівнів та для босів. Для звичайних рівнів просто ціль з дерева, а для босів різний набір дивних незвичних форм (на даному етапі в грі лише бос сир, але можна створювати префаби нових босів і просто додавати їх у список, а гра вже серед цього списку буде вибирати випадкові вигляди цілі для босів). Ціль обертається з рандомною швидкістю в діапазоні швидкостей і через певний час (також через рандомний проміжок часу із діапазону) змінює свій напрямок обертання в інший бік. Для босів логіка така ж сама, але швидкість обертання більша, а час зміни напрямку обертання менший (для складності). В рандомному місці на цілі генеруються яблука. За збирання яблука надається +3 бали до загального рахунку, а при звичайному попаданні меча в ціль +1 бал. При попаданні меча в ціль або яблуко з'являються частинки для візуалу, а сама ціль піднімається і швидко опускається по осі Y (імітація струсу) і на мить стає трохи світлішою. Додатково у босів при генерації рівня відразу з'являються від 1 до 3 мечів вставлених в ціль.

# Ножі
Всього в грі є 3 вигляди ножів, але про це описано в розділі "Магазин". Ножі, які вилітають мають свою точку спавну. При натраплянні меча на меч гравець програє (логіка рівнів в цьому випадку описана в розділі "Рівні"). З лівого боку є панель іконок ножів. По ній можна орієнтуватися скільки мечів треба викинути, щоб пройти раунд, скільки вже викинуто (іконка темніє) і скільки ще залишилося. На бахових рівнях кількість ножів, які необхідно викинути для переходу на наступний рівень, дорівнює 7, а для босів - 13.

# Рахунок
В системі є загальний рахунок, який не обнуляється з кожним новим рівнем, а накопичується і зберігається в пам'яті (тому при другому заході в гру це значення зберігається). Якщо гра відкрита вперше, то стартова сума очок = 50. Також в системі йде рахунок рівнів.

# Рівні
Коли користувач на певному рівні викидує всі мечі і вони всі успішно попали в ціль, то користувача переносить на наступний рівень. Якщо під час проходження звичайного рівня гравець програє, то цей самий рівень перезапускається (але яблука генеруються вже по іншому, бо там все працює на рандомі). Якщо ж гравець програє на рівні боса, то його відкотує на рівень останньої точки спавну. Точка спавну ставиться відразу на наступний рівень, який був після попередньго боса. Бос вмикається кожнен 5-тий рівень (5-ий, 10-ий, 15-ий ... тощо). Кнопка "Back" працює точно з такою ж логікою, що і програш, але й повертає на головну сторінку.

# Головне меню
Тут є кнопки Play, Shop, Quit та кнопка таблиці лідерів (про неї в розділі "Таблиця лідерів"). Кнопка Play запускає останній відкритий рівень, Shop - відкриває магазин, Quit - для виходу з гри.

# Таблиця лідерів
Вона визначається за максимальною кількістю зібраних поспіль яблук за один забіг (до першого програшу в грі). Всього ТОП-5 місць.

# Магазин
В магазині надано 3 вигляди ножів. Перший ніж завжди відкритий і доступний. 2 інші ще ні, тому і взаємодіяти з кнопками для вибору певного вигляду не можна. Для придбання нового вигляду треба мати >= значення в загальному score, порівняно із вартістю меча. значення score >= вартісті меча, то кнопка стає доступною для взаємодії та набуває оранжевого кольору. Як тільки користувач придбає новий вигляд меч у грі також змінить свій вигляд на той, що користувач щойно придбав. Якщо меч куплено, але як "скін" вибрано зараз інший, то колір кнопки білий і текст "owned". якщо меч вибрано як "скін", то кнопка набуває зеленого кольору із текстом "selected". При покупці меча від загального рахунку віднімається вартість меча.
